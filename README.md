# Examen Project Static Web: Telify

Telify is een webshop gemaakt enkel met HTML en CSS. 
Het biedt een aantal producten, met elk een specifieke pagina met verschillende afbeeldingen en karakteristieken.
Het site is volledig responsive en kan op mobiele devices gebruikt worden.


## Plan van aanpak


### **Maandag**
- **Namiddag**
  - Basisstructuur van de homepage maken
- **Avond**
  - Basis design van de homepage maken

### **Dinsdag**
- **Voormiddag**
  - Basisstructuur van subpagina’s voor specifieke producten
- **Namiddag**
  - Basis design van de subpagina’s voor specifieke producten
- **Avond**
  - Media queries voor de hele website maken

### **Woensdag**
- **Voormiddag**
  - Voor elke beschikbare producten (6) een subpagina aanpassen met verschillende reviews, info’s en beelden
- **Namiddag & Avond**
  - Commentaar toevoegen aan de HTML en CSS van de hele website

### **Donderdag**
- **Voormiddag**
  - Laatste details toevoegen, responsiveness checken
  - Commentaar toevoegen aan de HTML en CSS van de backend



## Externe bronnen

Chat-GPT werd gebruikt tijdens het maken van dit project.  

- Chat-GPT werd enkel gebruikt om kleine fixes te corrigeren, tips te krijgen over hoe ik een specifieke design kan maken in CSS, en om voorbeelden van reviews, karakteristieken en prijzen van smartphones te krijgen om het procesus te versnellen.    

**Geen enkele stuk code werd gegenereerd door AI in dit project**  

**Unsplash** werd gebruikt om copyright-free beelden te gebruiken voor alle producten.  
**Canva** werd gebruikt om de grootte van de verschillende afbeeldingen makkelijk te controleren en toe te voegen aan het project.
