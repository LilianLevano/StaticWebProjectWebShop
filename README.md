# Examen Project Static Web: Telify

Telify is een webshop gemaakt enkel met HTML en CSS. 
Het biedt een aantal producten, met elk een specifieke pagina met verschillende afbeeldingen en karakteristieken.
Het site is volledig responsive en kan op mobiele devices gebruikt worden.

## Plan van aanpak

### **Maandag**
- **Namiddag**
  - Basisstructuur van de homepage maken
- **Avond**
  - Basis design van de homepage maken

### **Dinsdag**
- **Voormiddag**
  - Basisstructuur van subpagina’s voor specifieke producten
- **Namiddag**
  - Basis design van de subpagina’s voor specifieke producten
- **Avond**
  - Media queries voor de hele website maken

### **Woensdag**
- **Voormiddag**
  - Voor elke beschikbare producten (6) een subpagina aanpassen met verschillende reviews, info’s en beelden
- **Namiddag & Avond**
  - Commentaar toevoegen aan de HTML en CSS van de hele website

### **Donderdag**
- **Voormiddag**
  - Laatste details toevoegen, responsiveness checken
  - Commentaar toevoegen aan de HTML en CSS van de backend

## Externe bronnen

Chat-GPT werd gebruikt tijdens het maken van dit project.  

- Chat-GPT werd enkel gebruikt voor het maken van de logo, voor kleine fixes (zoals een media querry die niet werkt door de specificiteit van een classe), tips te krijgen over hoe ik een specifieke design kan maken in CSS, en om voorbeelden van reviews, karakteristieken en prijzen van smartphones te krijgen om het procesus te versnellen.    

**Geen enkele stuk code werd gegenereerd door AI in dit project**  

**Unsplash** werd gebruikt om copyright-free beelden te gebruiken voor alle producten.  
**Canva** werd gebruikt om de grootte van de verschillende afbeeldingen makkelijk te controleren en toe te voegen aan het project.  

## Instructies

- Op de homepage, in de "check out" sectie staat er een "Dat wil ik zien!" button. Deze button werkt en stuurt u naar een pagina van het meest recente product.
- Op de homepage, elke product in de "grid" heeft een "meer details" product. Deze button werkt en stuurt u naar de pagina van deze product.
- Op de homepage, de (main) navigatie werkt en stuurt u naar de verschillende onderdelen van de pagina. Deze zelfde main navigatie zit ook op de subpagina's, maar sturen u gewoon op de homepage en niet specifiek naar zijn onderdelen
- Op de homepage heeft elke product en review een kleine effect wanneer het wordt gehovered
- De backend is niet responsief. Het kan breken op kleinere schermen.
- Op de backend, in de categorie "Actie" in de table, de groene knop edit werkt en stuurt u naar een HTML pagina om een product te "wijzigen".
